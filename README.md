# S.HOT – Spirulina Frontend

This is the frontend application for the S.HOT Spirulina brand, built with React and Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- [React 19](https://react.dev/)
- [Vite](https://vite.dev/) with [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) (Babel / Fast Refresh)
- [Tailwind CSS](https://tailwindcss.com/)
- [react-router-dom](https://reactrouter.com/)
- [react-i18next](https://react.i18next.com/) for internationalization

## GitHub Copilot – Interaction Data Policy

> **What does this notification mean?**
>
> GitHub announced that, starting **April 24**, it will begin using GitHub Copilot interaction data (such as prompts you type, code context sent to Copilot, and whether suggestions are accepted or rejected) to help train and improve its AI models — **unless you opt out**.

### What data is involved?

- Prompts and code snippets you send to Copilot while coding
- Copilot's suggestions and whether you accepted or dismissed them
- Editor context (language, file type, surrounding code)

### How to opt out

1. Go to your **GitHub account settings**: [https://github.com/settings/copilot](https://github.com/settings/copilot)
2. Under **"Policies"**, find the option **"Allow GitHub to use my data to improve GitHub Copilot"** (or similar wording).
3. Toggle it **off** to opt out before April 24.

### Does this affect this repository's source code?

No. This policy applies to each **individual GitHub user's** Copilot settings, not to the repository itself. Every contributor who uses GitHub Copilot should review their own account settings and decide whether they want to opt in or out.
